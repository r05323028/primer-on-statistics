# Primer on Statistics

這是一本用程式碼搭配基礎統計學的書。

## 動機

- 在統計學課本上常常會有一些定理很難理解，希望能藉由程式碼實際執行，來幫助大家更好地理解。

## 參考書籍

- *A First Course in Probability, Sheldon Ross*{cite:p}`ross2014first`
